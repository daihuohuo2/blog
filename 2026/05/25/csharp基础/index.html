C# 是什么

C# 读作 C Sharp，是一门由 Microsoft 推出的面向对象编程语言。

它常用于：

  • Windows 桌面程序开发
  • Web 后端开发
  • 游戏开发，比如 Unity
  • 命令行工具
  • 企业级管理系统
  • 跨平台应用开发

学习路线

建议按这个顺序学习：

  1. 输出、输入、变量
  2. 条件判断和循环
  3. 数组、List 集合
  4. 方法
  5. 类、对象、属性
  6. 异常处理
  7. 文件读写
  8. 面向对象进阶：继承、接口、多态
  9. 使用 .NET 开发控制台程序、Web 程序或 Unity 游戏

刚开始学习时，不需要一次记住所有语法。先能看懂代码，再能改代码，最后再自己写完整程序。
