## dirsearch 目录爆破工具(kali)

dirsearch 是一个 Web 目录和文件扫描工具，用于发现目标网站中隐藏的目录和文件

{% asset_img 1.png %}

```bush
# 最简单的用法
dirsearch -u http://目标网站

# 指定后缀名
dirsearch -u http://目标网站 -e php,html,js

# 指定字典
dirsearch -u http://目标网站 -w /usr/share/wordlists/dirb/common.txt

# 排除状态码
dirsearch -u http://目标网站 --exclude-status 404

# 只显示 200 状态码
dirsearch -u http://目标网站 --include-status 200

# 递归扫描
dirsearch -u http://目标网站 -r

# 设置线程数（加快速度）
dirsearch -u http://目标网站 -t 50

# 使用代理
dirsearch -u http://目标网站 --proxy http://127.0.0.1:8080
```
## Githack  源码泄露利用工具 (kali)

GitHack 是一个 Git 源码泄露利用工具，用于从目标网站下载泄露的 Git 仓库源码

什么是 Git 泄露

有些网站在部署时没有删除 .git 文件夹，导致攻击者可以通过访问 `http://目标网站/.git/` 获取网站源码。

使用的时候可以搭配dirsearch扫出带有.git的文件夹再使用GitHack进行源码还原

注意，写指令的时候不要把这个GitHack大小写搞混了

## fenjing (kali)

fenjing 是一个 SSTI（服务端模板注入）自动化检测和利用工具，专门用于绕过 WAF 进行模板注入攻击

注意，新版kali已经不允许直接使用pip了，需要自己创建虚拟环境

```bush
##安装

pip install fenjing

##启动webui
python -m fenjing webui

##使用方法

## 检测 GET 参数
fenjing scan --url "http://target.com/page?name=test"

## 检测 POST 参数
fenjing scan --url "http://target.com/page" --method POST --data "name=test"

# 交互式 Shell
fenjing exec --url "http://target.com/page?name=test"

```
